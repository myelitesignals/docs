# Do I Need a TradingView Account to Use EliteAlgo?

## EliteAlgo on TradingView

EliteAlgo is exclusively hosted on [**TradingView.com**](https://TradingView.com) and the [Chartlabs.co](https://Chartlabs.co) charting platforms.

### Getting Started

If you don't already have a TradingView account, simply visit [**TradingView.com**](https://www.tradingview.com) and create one. Once you're set up, you can sign up for **EliteAlgo** to enhance your trading experience.
