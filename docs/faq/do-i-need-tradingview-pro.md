# Do I Need TradingView Pro to Use EliteAlgo?

## EliteAlgo and TradingView

No need to upgrade — **EliteAlgo** works perfectly with the **free version** of TradingView.

The paid versions of TradingView offer additional features, such as:

- Setting alerts
- Using more than 3 indicators on one chart
- Accessing smaller timeframes
- And other premium features

However, the **free version** does not affect the performance of **EliteAlgo** in any way. You can use it to its full potential without any issues.
